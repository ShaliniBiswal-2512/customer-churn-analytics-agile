# Customer Churn Analytics using Agile (Jira + Power BI)

## Overview
End-to-end customer churn analysis project managed using Jira Scrum workflow.

Analyzed 7K+ customer records to identify churn drivers and provide business recommendations.

## Tools Used
- Jira (Agile Project Management)
- Power BI (Dashboard)
- Excel (EDA + Pivot)
- Kaggle Dataset

## Key Insights
- Overall churn rate: 26.54%
- Highest churn in early customer tenure
- Month-to-month contracts have highest churn
- Electronic check users churn most

## Business Recommendations
- Improve onboarding for first 90 days
- Encourage long-term contracts
- Promote automatic payments
